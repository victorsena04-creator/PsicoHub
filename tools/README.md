Esta pasta contém os scripts Python determinísticos.

### Regras

1. Scripts devem ser atômicos e testáveis
2. Variáveis de ambiente/tokens são armazenados em `.env`
3. Use `.tmp/` para todas as operações de arquivos intermediários
4. Scripts devem ser determinísticos (não probabilísticos)

### Estrutura Recomendada

- `README.md` - Este arquivo
- `*.py` - Scripts Python individuais
- `requirements.txt` - Dependências Python (se necessário)
